# water-buddy 项目规范

本文件与 `AGENTS.md` 保持一致，用于兼容仍读取 `CLAUDE.md` 的旧规则。

## 项目定位

Windows 喝水小助手，轻提醒、桌面常驻、简约可爱。第一优先级是长期使用不打扰，其次是视觉和动效让记录喝水这件事有一点愉悦感。

## 技术约定

- 使用 Python。
- 桌面 UI 使用 PySide6。
- 托盘、通知、动画优先使用 Qt 内置能力。
- 本地数据存到用户目录，不写入源码目录。
- 代码入口为 `src/water_buddy/main.py`。

## UI 约定

- 风格：简约、柔和、可爱，不做复杂装饰。
- 动效：优先使用非线性 easing，例如 OutBack、OutCubic、InOutSine。
- 提醒方式：轻提醒，不强制打断用户。
- 文案：中文、短句、少命令感。

## 禁止项

- 不把提醒做成全屏或强制置顶阻断操作。
- 不把用户配置写死在代码里。
- 不引入无必要的大型依赖。
- 不记录隐私数据、账号、token、密码。

## 任务归档

非平凡改动写入 `docs/` 下的工作日志，记录日期、做了什么、如何运行、未完成项。
